    Key Performance Optimizations in full Mode

     1. DB Direct Queries: Skip Python for tools/list, resources/list, prompts/list
     2. Plan Caching: Cache resolved tools/call plans (avoids repeated resolve requests)
     3. Session Reuse: Cache upstream MCP sessions (avoid re-initialize)
     4. Auth Context Reuse: Cache auth context per session (skip Python auth round-trip)
     5. Affinity Routing: Direct session-owner forwarding (avoid broadcast)
     6. Event Store: Persist session state for resume (survive restarts)
